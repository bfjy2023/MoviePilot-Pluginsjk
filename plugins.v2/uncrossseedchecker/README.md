# 未辅种检查器

MoviePilot 插件，用于检查下载器中指定站点的种子是否已辅种到其他站点。适合在执行 IYUU 辅种后，找出哪些种子还没有辅种到目标站点，方便手动去发种。

## 功能特性

- 扫描下载器全部种子，通过 tracker URL 自动识别站点
- 按种子内容去重（相同大小 + 名称视为同一内容），展示实际内容数量
- 对比指定目标站点，列出已有和未有该站点的内容
- 左右双栏布局，固定高度可滚动
- 站点名称显示为可点击芯片，点击跳转到对应站点种子详情页
- 支持 qBittorrent 和 Transmission 下载器
- 支持配置最小种子大小过滤
- 支持通知推送和 TG Bot 命令

## 支持版本

- V2 插件目录：`plugins.v2/uncrossseedchecker`
- V2 索引：`package.v2.json`

## 安装说明

1. 在 MoviePilot 中进入插件仓库设置，将当前 GitHub 仓库地址添加为插件仓库。
2. 保存并刷新插件市场。
3. 在插件市场安装"未辅种检查器"。
4. 进入插件配置页，选择下载器和要检查的站点。
5. 启用插件后，点击"立即运行一次"执行检查。

## 配置项说明

| 配置项 | 说明 |
| --- | --- |
| 启用插件 | 开启插件 |
| 发送通知 | 检查完成后发送通知 |
| 立即运行一次 | 保存配置后后台执行一次，并自动关闭该开关 |
| 下载器 | 选择要检查的下载器（单选） |
| 检查站点 | 选择要检查的目标站点（单选） |
| 最小种子大小 (GB) | 过滤掉小于此大小的种子，0 表示不过滤 |

## 使用流程

1. 先运行 IYUU 辅种工具完成辅种
2. 打开未辅种检查器插件
3. 选择下载器和目标站点
4. 点击"立即运行一次"
5. 查看结果：
   - **未有该站点**：这些内容在目标站点上还没有种子，可以去发种
   - **已有该站点**：这些内容已经辅种到目标站点

## 结果说明

- **下载器种子数**：下载器中的原始种子总数
- **去重后内容数**：按内容去重后的实际数量（相同大小 + 名称视为同一内容）
- **已有该站点**：目标站点已有的内容数量
- **未有该站点**：目标站点还没有的内容数量（可去发种）
- **站点芯片**：点击可跳转到对应站点的种子详情页

## 常见问题

### 没有显示种子列表

请确认已选择下载器和目标站点，然后点击"立即运行一次"执行检查。

### 站点识别不准确

插件通过 tracker URL 的域名匹配站点。如果站点使用了不同的 tracker 域名，可能无法识别。可以在站点管理中确认站点的域名配置。

### 显示 localhost 链接

部分种子的 comment 字段可能是 localhost 链接（本地添加的种子），插件会自动过滤这些无效链接。

### 如何检查多个站点

目前只支持单选站点。需要检查多个站点时，切换站点后重新运行即可。
